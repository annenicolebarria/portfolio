Project 1

Birthday Website


This project is a simple yet creative website designed with sections including Home, About, Services, Portfolio, and Contact. It features a clean container-based layout, personalized greetings, and additional links to make navigation more engaging. A toggle function was also implemented for smoother browsing and user interaction. The website is fully responsive, ensuring that it works seamlessly across different devices such as desktops, tablets, and mobile phones. Through this project, I was able to strengthen my foundation in HTML, CSS, and JavaScript, while practicing responsive web design, proper content structuring, and enhancing user experience. It served as an important starting point in my journey into web development, combining both functionality and presentation
